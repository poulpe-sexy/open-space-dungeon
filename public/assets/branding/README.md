# Branding assets

Les images de marque du jeu (logo du titre, etc.).

## `title-logo.png`

Logo pixel art « OPEN SPACE DUNGEON » affiché sur l’écran d’accueil ET en
intro. Recommandé : PNG à fond transparent, largeur native ≥ 680 px (le
composant CSS le contraint à max 560 px en rendu, avec
`image-rendering: pixelated`).

Si le fichier est absent, le `TitleLogo` tombe automatiquement sur le texte
stylé « OPEN SPACE DUNGEON » — le jeu reste jouable.
